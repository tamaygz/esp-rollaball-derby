Usage Notes

Developer

HTTP Path

The Express server mounts clients/assets/ at /assets/.

  • Horse sprite: /assets/themes/horse/sprite.svg
  • Camel sprite: /assets/themes/camel/sprite.svg
  • Track background: /assets/themes/{theme}/track-bg.svg
  • Finish flag: /assets/themes/{theme}/finish-flag.svg
  • Theme metadata: /assets/themes/{theme}/theme.json
  • Player colours: /assets/themes/shared/player-colors.json

Pixi.js Tinting

All sprites use pure white fill (fill="white"). Load as a Pixi Texture, create a Sprite, then:

sprite.tint = 0xE53E3E; // player colour index 0 = Scarlet

Pixi multiplies each pixel channel by tint/255. White × tint = exact tint colour. Grey shades (jockey helmet ~85% white) render at 85% of tint brightness — natural shading preserved.

Theme Loading

Load theme.json to get player colour array, palette, and asset paths. Example:

const theme = await fetch('/assets/themes/horse/theme.json').then(r => r.json());
const tint = parseInt(theme.playerColors[playerIndex].pixi, 16);

Animation

Sprites are static (single frame). Simulate animation via:

  • Horizontal position: smooth tween via gsap.to(sprite, { x: targetX, duration: 0.4, ease: 'power2.out' })
  • Score bounce: gsap.timeline().to(sprite, { y: '-=8', duration: 0.12 }).to(sprite, { y: '+=8', duration: 0.18, ease: 'bounce.out' })
  • Idle bob: gsap.to(sprite, { y: '+=3', duration: 0.8, yoyo: true, repeat: -1, ease: 'sine.inOut' })
