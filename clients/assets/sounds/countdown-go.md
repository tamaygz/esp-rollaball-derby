# countdown-go

The "GO!" moment — race is starting. Energetic, uplifting, explosive start signal.

## Horse Derby

> Bright brass fanfare blast with a starting bell ring, like a horse race starting gate opening. Triumphant ascending three-note brass burst. 0.5 seconds, 44100 Hz stereo.

## Camel Derby

> Loud Arabian horn blast with a crash cymbal, like a desert race starting signal with shimmering metallic overtones. Energetic ascending melody on a zurna. 0.5 seconds, 44100 Hz stereo.
