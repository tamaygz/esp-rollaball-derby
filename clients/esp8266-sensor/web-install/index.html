Derby Sensor Flasher

Flash the IR sensor firmware to a Wemos D1 Mini — no toolchain required.

⚠️ Unsupported browser. Web Serial requires Chrome or Edge 89+ on desktop. Safari and Firefox are not supported.
  1. Plug the Wemos D1 Mini into a USB port.
  2. Click Install and select the serial port.
  3. Wait ~30 s for flashing to complete.
  4. On first boot, connect to the Derby-Sensor-XXXX WiFi AP and enter your server IP & port.
Install Derby Sensor Firmware

Downloads the latest pre-built firmware from GitHub Releases. To build from source, see the sensor README.
